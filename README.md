# BriefLoop — Executive Prep & Follow-Up Copilot

BriefLoop is a privacy-safe proof-of-concept tool for executive operations.

It demonstrates:
- meeting prep brief generation
- relationship context retrieval
- open-loop tracking
- post-meeting follow-up extraction
- local-first LLM use through vLLM/Qwen

## Privacy boundary

This project uses only synthetic data. It does not use real CEO data, real donor data, real grantee data, real inboxes, real calendars, or private relationship history.
